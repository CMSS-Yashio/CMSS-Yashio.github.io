理数研究部とは





　理数研究部は品川区にある都立校の都立八潮高等学校の研究部の一つです。
　活動する曜日は、毎年、部員で相談して決めています。今年は水曜日・金曜日に活動しています。自分の出られる曜日だけ活動している部員もいます。活動場所としては主に物理実験室を使い、専用倉庫も別にあります。


　理数研究部だからといって文系が少ないわけではありません。数学や理科が好きという人で集まって研究や実験を行っています。また、数学や理科が苦手だからと言って心配することはありません。部員相互での勉強会や、先輩方・先生方による特別講義が行われているので、初歩的な内容から高度な内容まで、じっくりと学ぶことができます。


　ドラフトが完備されている化学実験室などの各種実験室を始めとして、充実した施設を使用することができます。先輩方や先生方の協力で、大学などの研究機関にある高度な施設も使用することができます。
　実験室や専用倉庫には多くの実験器具が用意されています。もちろん自分で持っていたり、特殊な実験をするために器具を個人的に買ったりした場合にはそれを置いておくための棚も用意されています。先輩方が置いていってくださった実験器具も数多くあります。


専用倉庫内の本棚には多くの科学関連書籍が置いてあります。実験書もいつでも閲覧できる状態となっています。他にも過去の部誌やニュートンなどの資料が置いてあり、気になることはインターネットに頼らなくてもすぐに調べられる状態になっています。実験には常に危険が隣り合わせですので、安全対策としての手袋や白衣、安全眼鏡も自由に使え、救急箱も完備しています。
